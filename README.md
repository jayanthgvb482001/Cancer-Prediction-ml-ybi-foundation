# Cancer Prediction using Machine Learning

## Project Overview
This project focuses on predicting whether a cancer tumor is Malignant or Benign using Machine Learning algorithms. The model is trained using the Breast Cancer dataset and compares different ML algorithms to achieve better prediction accuracy.

## Dataset
Dataset: Cancer.csv

The dataset contains different features related to cell measurements such as:
- Radius
- Texture
- Perimeter
- Area
- Smoothness
- Compactness
- Symmetry

Target Variable:
- Diagnosis (Malignant / Benign)

## Machine Learning Algorithms Used
The following algorithms are implemented and compared:

1. Logistic Regression
2. Decision Tree Classifier
3. Random Forest Classifier
4. Support Vector Machine (SVM)
5. K-Nearest Neighbors (KNN)

## Project Workflow

1. Import required libraries
2. Load dataset
3. Data preprocessing
4. Split data into training and testing sets
5. Train Machine Learning models
6. Compare model performance
7. Predict cancer diagnosis

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab / Jupyter Notebook

## Results

Different Machine Learning models are evaluated using accuracy score to identify the best performing model for cancer prediction.Support Vector Machine (SVM) achieved the highest accuracy among all implemented models and was selected as the best model for cancer prediction.

## Conclusion

Machine Learning techniques can help in early cancer prediction by analyzing medical data. The comparison of multiple algorithms helps select an effective model.
