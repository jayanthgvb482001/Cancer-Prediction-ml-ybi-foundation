# Cancer Prediction Using Machine Learning

## Project Overview

This project focuses on predicting cancer using Machine Learning classification algorithms. The dataset is analyzed and processed to train multiple machine learning models and compare their performance to identify the best model for cancer prediction.

## Objectives

- To build a machine learning model for cancer prediction.
- To analyze and preprocess the cancer dataset.
- To compare different ML algorithms based on performance.

## Dataset

The dataset used in this project is the Cancer Dataset from YBI Foundation.

Dataset Source:
YBI Foundation Dataset Repository

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

## Machine Learning Algorithms Used

The following classification algorithms were implemented:

1. Logistic Regression
2. Decision Tree Classifier
3. Random Forest Classifier
4. K-Nearest Neighbors (KNN)
5. Support Vector Machine (SVM)

## Project Workflow

1. Import required libraries
2. Load cancer dataset
3. Analyze dataset information
4. Data preprocessing
5. Split data into training and testing sets
6. Train multiple ML models
7. Evaluate model accuracy
8. Compare model performance

## Results

Multiple machine learning algorithms were trained and evaluated for cancer prediction.

Support Vector Machine (SVM) achieved the best performance among the implemented models.

The comparison of different algorithms helps in selecting the most suitable model for accurate cancer prediction.

## Conclusion

This project demonstrates how machine learning techniques can be applied to healthcare data. By comparing different classification models, the most effective algorithm can be identified for predicting cancer.
